Uso da API

Esta seção descreve os serviços disponíveis através da API do Controle Condomínio APP.

Tenha em mente que apenas os metodos GET funcionam utilizando o browser, para os demais metodos é necessario usar outros recursos como o POSTMAN

>

Classes e Serviços

Morador

GET /moradores/lista: Retorna uma lista de todos os moradores.

POST /moradores/incluir: Inclui um novo morador.

DELETE /moradores/:id/excluir: Exclui um morador pelo ID.

GET /moradores/:id: Retorna os detalhes de um morador pelo ID.

Veículo
